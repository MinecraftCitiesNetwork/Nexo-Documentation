# ðŸª‘ Seat Mechanic

Using the Seat-Mechanic you can add seats to your furniture.\
Seats are configured like below, with an `x,y,z` offset

```yaml
myitem:
  Mechanics:
    furniture:
      seats:
        - 0,0.5,0
```
